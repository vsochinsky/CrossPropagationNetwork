Небольшое пояснение к программе

map.txt - файл с описанием сенсоров. Сенсоры указываются с новой строки в
формате:
x1 y1 x2 y2 N
где
  x1, y1 - координаты правой верхней вершины
  x2, y2 - координаты левой нижней вершины (предполагается x2 > x1 ; y2 > y1)
  N - количество секторов, на которое делится сторона сенсора при подсчёте
  среднего показания
Все сенсоры предполагаются прямоугольными, с размерми, не превышающими размера
карты

out.txt - обработанный лог, формат совпадает с форматом исходного лога,
исходный лог оставлен без изменений под прежним именем. В конце
каждой строки добавлены показания сенсоров, в порядке их указания в файле map.txt
Диапазон значений, выдаваемых сенсором - от 0 до 255. 0 соответствует минимальной
высоте, 255 - максимальной

templog.txt - временный файл программы, используется в процессе её работы.
Смысловой нагрузки не несёт. Не предполагает удаления/изменения вручную

main_win32.exe - исполняемый файл программы

main.c - исходный код программы

Остальные файлы, относящиеся к предыдущей программе, оставлены без изменений


После завершения работы программы для выхода введите любой символ и нажмите
Enter, или закройте окно программы ( сделано для того, чтобы можно было
прочитать вывод отладочной информации на терминал )
